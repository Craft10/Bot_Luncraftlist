API for Luncraftlist!

This is a simple API REST for Luncraftlist, you can use it to get information about bot

https://localhost:8000

  • /api/bots Obtiene Todos los bots este aceptados o no.
  • /api/bot/:id Obtiene la informacion de un bot por su ID.
  • /api/bots Obtiene los Bots que estan aceptados.
  • /api/ad Obtiene Todos los Bots que son Anuncios.
  • /api/votescount/:bot_id Obtiene Todos los Votos en Total.
  • /api/bot_owner/:id Obtiene Todos los bots de un UserId
